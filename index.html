    MicroPython¶

    Reference¶

    Basic steps to get started with MicroPython:

    1. Flash MicroPython firmware on ESP32 microcontroller
    2. Upload code with ampy
    3. Monitor from serial console

    Use WebREPL for remote serial connectivity
